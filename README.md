# Ship Shooter Game

A simple 2D ship shooting game built with **Python** and **Pygame**.  
Move your ship left and right and shoot bullets to defend yourself (or just for fun!).  

---

## **Features**

- Move the ship left and right using arrow keys.  
- Fire bullets using the spacebar.  
- Bullets disappear when they leave the screen.  
- Simple and lightweight â€” great for beginners learning Pygame.  

---

## **How to Play**

1. **Move the ship**:  
   - Press **Left Arrow** to move left.  
   - Press **Right Arrow** to move right.  

2. **Shoot bullets**:  
   - Press **Spacebar** to fire a bullet.  

3. **Goal**:  
   - Currently, there are no enemies, so the main goal is to practice movement and shooting.  
   - You can expand the game by adding enemies, scoring, and levels.
